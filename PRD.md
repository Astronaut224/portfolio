# 个人作品集网站 PRD

## 核心功能
1. 首页：大标题 + 简介 + 头像
2. 关于我：详细介绍 + 技能列表
3. 项目展示：项目卡片列表，每个卡片包含项目名称、项目截图、简短描述、技术栈、项目链接
4. 联系方式：邮箱、GitHub、社交媒体链接

## 设计要求
- 双主题，默认深色，支持用户切换并记忆选择（localStorage）
- 浅色模式（极简苹果风）：纯白背景 #FFFFFF、纯蓝主色 #007AFF、系统灰辅助色、极浅弥散阴影、毛玻璃卡片、品牌字纯蓝实色无渐变
- 深色模式（未来赛博风）：深空黑背景 #0A0A0A、电光青主色 #00F0FF、霓虹紫点缀、青蓝渐变品牌字、卡片青色描边、霓虹发光阴影、青色滑块滚动条
- 流畅的滚动进入动画与卡片级延迟渐入
- 移动端适配
