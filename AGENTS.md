# 个人作品集网站开发指令

## 项目概述
使用 Vue + TypeScript + Tailwind CSS 开发的个人作品集网站。

## 开发环境
- 操作系统：Windows
- 终端：PowerShell
- 请使用 Windows 兼容的命令

## 开发规范
- 使用函数式组件 + Hooks
- 使用 Tailwind CSS 编写样式
- 组件要可复用
- 代码要有注释

## 设计要求
- 双主题，默认深色，支持用户切换并记忆选择（localStorage）
- 浅色模式（极简苹果风）：纯白背景 #FFFFFF、纯蓝主色 #007AFF、系统灰辅助色、极浅弥散阴影、毛玻璃卡片、品牌字纯蓝实色无渐变
- 深色模式（未来赛博风）：深空黑背景 #0A0A0A、电光青主色 #00F0FF、霓虹紫点缀、青蓝渐变品牌字、卡片青色描边、霓虹发光阴影、青色滑块滚动条
- 颜色与阴影统一通过语义 token 管理（src/style.css 中 @theme 浅色默认 + .dark 覆盖深度）
- 添加平滑的滚动动画与卡片级延迟渐入
- 确保移动端适配

## 注意事项
- 保持设计简洁，不要过度设计
- 性能优化：图片使用懒加载
- 确保所有链接可点击
